  4. Utredning når diagnosen er stilt

Mål for utredningen

Mål for utredningen er å kartlegge:

  • hvilken type diabetes som foreligger
  • sykdommer eller andre forhold som påvirker glukosetoleranse (medikamenter, infeksjon, annen alvorlig sykdom; videre utredning ved klinisk mistanke)
  • diabeteskomplikasjoner (ved type 2­diabetes kan det foreligge komplikasjoner ved diagnosetidspunktet)
  • risiko for hjerte­ og karsykdom
  • pasientens psykososiale situasjon; denne påvirker mestring av behandlingskrav i hverdagen (mental helse, sosial støtte, ressurser)
  • basisregistrering for å lage en behandlingsplan, vurdere behandling og utviklingen ved senere kontroller

Bruk av strukturert elektronisk diabetesjournal (for eksempel NOKLUS­Diabetes fagsystem i sykehusene og NOKLUS diabetesskjema i primærhelsetjenesten) forenkler og systematiserer oppfølgingen av personer med diabetes. Det gir også mulighet til å få en oversikt over kvaliteten på diabetesbehandlingen på henholdsvis lege­ og legesenter og avdelingsnivå. Bruk av skjemaet er også viktig for å rapportere data til Norsk Diabetesregister for voksne (www.noklus.no).

Innhold i utredningen

Anamnese

Legg spesiell vekt på:

  • symptomer (tørste, vekttap, øyesymptomer, infeksjoner, nedsatt allmenntilstand) og deres varighet
  • symptomer på eller kjent hjerte­ og karsykdommer i familien (diabetes, hjerte­ og karsykdom)
  • livssituasjon (familie, arbeid, utdanning)
  • kosthold (se skjema for kostanamnese, kapittel 15.4)
  • levevaner (fysisk aktivitet, røyking, vektendring osv.)
  • medikamentbruk
  • psykisk helse og ressurser nødvendig for mestring av sykdommen
  • obstruktiv søvnapnoe
  • ved type 1­diabetes: andre autoimmune sykdommer som tyreoideasykdommer, cøliaki, binyrebarksvikt og vitamin B12­mangel

Klinisk undersøkelse

Det bør gjøres en vanlig klinisk undersøkelse. Spesielt bør undersøkes:

  • høyde/vekt og livmål
  • blodtrykk
  • tegn på arteriosklerotisk sykdom
  • undersøkelse av føtter (se kapittel 8.7 og 11.9)
  • tilleggsundersøkelser ved mistanke om infeksjoner/annen bakenforliggende sykdom

Laboratiorieundersøkelser

  • B­Hb A1c
  • S­lipider (totalkolesterol, HDL­ og LDL­ kolesterol, fastende triglyserider)
  • estimert GFR eller S­kreatinin
  • S­TSH (overhyppighet av thyreoideasykdommer ved type 1­diabetes)
  • urinundersøkelse (glukose, teststrimler for nitrit og leukocytter, mikroalbuminuri/urin­albumin/kreatinin­ratio (U­AKR)) (se kapittel 14.6)
  • EKG (pasienter > 50 år eller ved høy risiko for hjerte­ og karsykdom)
  • antistoffer mot vevstransglutaminase ved type 1­diabetes (økt risiko for cøliaki)
  • evt. tilleggsundersøkelser med tanke på bakenforliggende sykdom
  • vurder behov/indikasjon for analyse av C­peptid, anti­GAD og eventuelt IA­2 (se kapittel 14.4 og 14.5)

Ved alder over 75­-80 år bør man vurdere indikasjon for analyse av lipider og U­ AKR ut fra den kliniske situasjonen og relevans for forebygging av senkomplikasjoner.

Differensialdiagnostikk type 1-/type 2-diabetes og annen diabetes

Type 1-­diabetes oppstår hyppigst hos barn, ungdom og unge voksne, men kan debutere i alle aldre. Type 2­diabetes oppstår oftest hos voksne og eldre, men forekommer nå i økende grad hos unge voksne. Hos ungdom er sykdommen foreløpig sjelden, selv ved ikke­vestlig etnisk bakgrunn. I enkelte land er forekomsten av type 2­diabetes hos ungdommer like høy som for type 1­ diabetes. Hos eldre overvektige er det som regel ikke nødvendig å utrede nærmere hvilken type diabetes som foreligger, dersom ikke blodglukose er svært høy (> 30 mmol/l).

Det kan noen ganger være vanskelig å avgjøre om det foreligger type 1­ eller type 2­diabetes. Følgende taler for type 1­diabetes: ketonuri, betydelig vekttap før diagnose, og fravær av metabolsk syndrom. Vurdér muligheten for type 2­ diabetes hos yngre pasienter dersom det foreligger betydelig bukfedme, mange tilfeller av diabetes i familien, eller ikke­vestlig bakgrunn. Vurdér måling av autoantistoffer (anti­GAD og IA2) samt C­peptid for å skille mellom type 1­ og type 2­diabetes. Husk at opptil 3 % av pasienter med diabetes kan ha monogen diabetes og at diabetes kan oppstå sekundært til annen sykdom (se kapittel 7.7 og kapittel 13, 14.4 og 14.5).

Mistanke om type 1-diabetes

Ved mistanke om nyoppstått type 1­diabetes bør pasienten henvises til spesialist/sykehus for videre utredning og start av insulinbehandling. Barn og unge henvises som øyeblikkelig hjelp, det samme er tilfelle for voksne med tegn til ketoacidose. Pasienter med LADA kan som regel behandles i primærhelsetjenesten.

Svangerskap og prevensjon

Kvinner med kjent diabetes trenger sikker prevensjon fordi svangerskap som ikke er planlagt kan gi økt risiko for abort og misdannelser. I utgangspunktet kan alle prevensjonsformer brukes, men sikkerheten vil veie ekstra tungt ved valg av prevensjonsmiddel for disse kvinnene. For yngre kvinner som ikke har født, anbefales primært lavdose kombinasjonspille. For kvinner > 35 år uten hjerte­ karsykdom kan gestagenpille være et alternativ, selv om sikkerheten er dårligere. IUD (spiral) er like effektiv hos kvinner med diabetes som hos ikke­ diabetikere, og er et godt alternativ for kvinner som har født barn. Hormonspiralen er svært sikker og kan i tillegg beskytte mot forstadier til endometriecancer, en kreftform som har økt forekomst ved type 2­diabetes.

Kvinner med kjent diabetes bør planlegge graviditet nøye. Man bør vurdere å henvise til prekonsepsjonell veiledning, helst ved den avdeling der kontrollen av diabetessvangerskapet skal finne sted. De bør uansett henvises dit så tidlig som mulig i svangerskapet. Ved diabetes som oppstår under svangerskapet (svangerskapsdiabetes) er henvisnings­ og oppfølgingsrutiner avhengige av blodglukosenivå (se kapittel 12).

Henvisning til øyelege

Siden retinopati kan foreligge allerede ved diagnosetidspunktet (5­15 %), bør alle pasienter med nyoppdaget type 2­diabetes henvises til øyelege. Pasienter med type 1­diabetes bør henvises etter 5 års diabetesvarighet, barn henvises første gang ved 10 års alder (se kapittel 11.6).

Diabetes og andre sykdommer

Mange pasienter med akutt hjerte­ og karsykdom har udiagnostisert diabetes eller nedsatt glukosetoleranse (39). Type 2­diabetes kan i sjeldne tilfelle oppstå sekundært til endokrine sykdommer (Mb. Cushing, akromegali, feokromocytom) eller utløses av andre sykdommer som alvorlige infeksjoner, pankreatitt, hemokromatose og malignitet. Type 1­diabetes kan opptre sammen med cøliaki, thyreoideasykdommer, Addisons sykdom og pernisiøs anemi. Det er beskrevet en prevalens av cøliaki hos barn med type 1­diabetes på 3­5 %, mot 0,3­1 % i befolkningen forøvrig. Det anbefales å screene pasienter med type 1­ diabetes med tanke på cøliaki ved diagnosetidspunktet og deretter hvert 5. år eller ved klinisk mistanke (se kapittel 7.2.3).
